# Clarionmark Group Website

This is a ready-to-use static website for Clarionmark Group.

## How to use locally
1. Unzip the package.
2. Open `index.html` in your browser.

## Deploy on GitHub Pages
1. Sign in to GitHub as `Trodm`.
2. Create a new repository named `clarionmark-group-website`.
3. Upload `index.html` and `styles.css` from this folder.
4. Go to **Settings > Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select branch **main** and folder **/root**.
7. Click **Save**.
8. GitHub will provide your live website link after publishing.
